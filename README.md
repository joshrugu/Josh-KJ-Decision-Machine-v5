# ❤️ The Josh & KJ Decision Machine V5

Mobile-first pastel PWA with:
- Built-in generated background music, no MP3 needed
- Sound effects for clicks, spin, win and reset
- Music tries to start when entering the app
- If iPhone/Chrome blocks autoplay, first tap anywhere starts music
- Activities/Food modes
- Filters
- Restaurant Google Maps buttons
- Regular food opens nearby search
- Memory album with camera/photo upload, notes and rating
- Reset everything
- Offline support
